# Netflix Data Analysis and Visualization

This project analyzes Netflix content data using Python for data cleaning, exploratory data analysis, and visualization.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Folium, KaggleHub, Requests

## Project Highlights
- Loaded and cleaned Netflix dataset from Kaggle
- Checked missing values and converted date fields
- Visualized content type, ratings, release years, and top countries
- Built a choropleth map to show the global distribution of Netflix titles

## Visualizations
- Content type distribution
- Content rating distribution
- Release year distribution
- Boxplot of release year by content type
- Top 10 countries by content type
- Global choropleth map of Netflix titles by country

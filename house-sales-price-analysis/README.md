# House Sales Price Analysis and Prediction

This project explores housing sales data and builds regression models to predict house prices. The analysis includes data cleaning, exploratory data analysis, model development, and model evaluation.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Overview
- Cleaned and prepared the dataset for analysis
- Performed exploratory data analysis to understand relationships between features and house price
- Built and compared different regression models, including:
  - Simple Linear Regression
  - Multiple Linear Regression
  - Ridge Regression
  - Polynomial Regression / Pipeline Model
- Evaluated model performance using $R^2$

## Key Insights
- Living area and above-ground square footage show positive relationships with house price
- Using multiple features improves prediction compared with a single-variable model
- Models with polynomial features performed better than the basic regression models on the test data

## Files
- `house_sales_price_analysis.ipynb` — main notebook

## Dataset
The dataset contains housing sales information, including variables such as price, bedrooms, bathrooms, square footage, location, and other housing characteristics.

## Goal
The goal of this project is to understand which housing features affect price and to develop models that can better predict house prices.
